# 更新日志

本项目遵循[语义化版本](https://semver.org/lang/zh-CN/)规范。

## [0.2.2] - 2026-07-15

### 主要功能
- 今日分组：优化带截止时间待办的今日智能分组展示
- 截止时间：重写应用内时间选择器，避免 Tauri 窗口内弹层越界，并支持确认后保存
- 分组视觉：调整未完成分组标题字号、字重、对齐方式和中英文字体一致性
- 任务布局：将截止时间状态移动到任务标题下方，减少右侧空间占用

### 发布说明
- `v0.2.2` 为 LightTodo 的待办列表体验优化版本
- 在 `v0.2.1` 的 Markdown 渲染基础上，重点完善今日分组、截止时间选择和分组标题视觉
- 本版本未改变同步协议和数据结构，主要为桌面端交互与布局优化

## [0.1.0] - 2026-06-29

### ✨ 新增功能
- 基础待办 CRUD（创建、编辑、删除）
- 待办完成状态切换
- 三级优先级标记（高🔴、中🟡、低⚪）
- 按优先级自动排序显示
- 窗口置顶功能
- 自动保存（失焦后触发）
- 空待办自动删除
- 本地 SQLite 数据库存储
- 响应式 UI 界面

### 🐛 问题修复
- 修复中文输入法（IME）问题（通过禁用窗口透明）
- 修复 React key 警告导致的界面错乱
- 修复新建待办后自动聚焦功能
- 修复按钮点击在拖拽区域失效的问题

### ⚙️ 技术实现
- 前端：React 18 + TypeScript + Tailwind CSS
- 后端：Rust + Tauri 2.0 + SQLite
- 状态管理：Zustand
- 构建工具：Vite

### 📝 文档
- 添加完整的 README 开发指南
- 添加项目结构说明
- 添加环境搭建文档

### ⚠️ 已知限制
- 窗口透明效果已禁用（为保证中文输入正常工作）
- 待办项置顶功能尚未实现（仅支持窗口置顶）
